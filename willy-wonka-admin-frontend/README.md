# Willy Wonka Admin Frontend

Фронтенд для сотрудников (админка)
- Реализованы все CRUD-операции для задач, пользователей, цехов, оборудования
- MUI (Material UI), современный интерфейс
- Язык интерфейса: русский
- Используется JWT авторизация
- Минимальные требования: Node.js 18+

## Запуск
```bash
npm install
npm start
```

## Основные разделы:
- Вход / Регистрация
- Сотрудники (пользователи)
- Задачи
- Оборудование
- Цеха
- Личный кабинет
